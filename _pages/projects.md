## Real-Time Expense Fraud Detection System
*Flask, PostgreSQL, Docker, Redis*

A comprehensive fraud detection system inspired by Ramp's prevention-first approach to expense management.

**Key Features:**
- Multi-layered anomaly detection with configurable rule engines
- Real-time transaction processing with sub-500ms response times
- Machine learning-powered pattern recognition
- RESTful API for integration with existing financial systems
- Comprehensive logging and audit trail

**Technical Highlights:**
- Microservices architecture using Flask and SQLAlchemy
- Asynchronous task processing with Celery and Redis
- Containerized deployment with Docker
- PostgreSQL for persistent storage with optimized queries

## Interfere - Self-Healing API System
*TypeScript, Express, Docker*

An intelligent system that automatically detects and resolves API schema mismatches.

**Key Features:**
- Automatic detection of API schema drift and mismatches
- LLM-powered analysis for intelligent issue diagnosis
- Canary deployment for safe, automated fixes
- Real-time monitoring and alerting

**Technical Highlights:**
- Built with TypeScript for type-safe development
- Express.js backend with modular architecture
- Integration with OpenAI/Claude for intelligent diagnostics
- Containerized microservices with Docker Compose
